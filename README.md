# VOLTEX – Electronics & Gadgets

A modern electronics and gadgets e-commerce frontend website built using HTML, CSS, and JavaScript.

## Features
- Product search system
- Category filtering
- Brand filtering
- Price range filter
- Wishlist functionality
- Add to cart system
- Product sorting
- Responsive futuristic UI
- Animated product cards and notifications

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Project Type
Frontend E-Commerce Website
